## 簡単な計算をする
dentaku.html

フォームから値を取得<br>
$(セレクタ).val()

フォームに値を出力<br>
$(セレクタ).val(値)

DOMからHTMLを取得<br>
$(セレクタ).html()

DOMにHTMLを出力<br>
$(セレクタ).html(値)

DOMからテキストを取得<br>
$(セレクタ).text()

DOMにテキストを出力<br>
$(セレクタ).text(値)
